Table library_spectra_meta

Table to store the meta data for each file (details are intended to be either extracted from an MSP file generated from either MassBank (https://github.com/MassBank/MassBank-data) or MoNA (http://mona.fiehnlab.ucdavis.edu/downloads). For associated regexes see https://github.com/computational-metabolomics/msp2db/blob/master/msp2db/re.py

IndexesField NameData TypeDescription
id integer
  name text The name of the spectra. Typically contains the name of the compound but can include other relevant information
  collision_energy text Collision energy used
  ms_level real Mass spectrometry level (e.g. 1, 2, 3)
* accession text The accession as determined from either MassBank or MoNA. This should be a unique identifier for the associated database the spectra was obtained from
  resolution text Resolution of the mass spectrometer
  polarity integer Polariity used for acquisition (e.g. positive or negative)
  fragmentation_type text Fragmentation type used e.g. HCD, CID
  precursor_mz real The m/z value of the precursor fragmented
  precursor_type text The precursor adduct type (e.g. [M-H]-, [M+H]+, [M+H-H2O]+ etc)
  instrument_type text Instrument type (ideally in the MassBank naming convention e.g. 'CE-ESI-TOF', 'ESI-ITFT', 'ESI-ITTOF', 'ESI-QTOF', 'LC-ESI-IT', 'LC-ESI-ITFT', 'LC-ESI-ITTOF', 'LC-ESI-QFT', 'LC-ESI-QIT', 'LC-ESI-QQ', 'LC-ESI-QTOF', 'LC-ESI-TOF', etc
  instrument text The mass spectrometry instrument name
  copyright text The copyright of the original spectra
  column text The chromatography column used
  mass_accuracy real The mass accuracy of the mass spectrometry instrument
  mass_error real The mass error of the mass spectrometry instrument
  origin text The origin of the individual spectra (sometimes can have a reference to the original file prior to creating the msp)
  splash text SPLASH database spectral identifier https://splash.fiehnlab.ucdavis.edu/
* library_spectra_source_id integer library_spectra_source id
* inchikey_id text inchikey id
Indexes
pk_library_spectra_meta ON id
Foreign Keys
Fk_library_spectra_meta ( library_spectra_source_id ) ref library_spectra_source (id)
Fk_library_spectra_meta ( inchikey_id ) ref metab_compound (inchikey_id)
