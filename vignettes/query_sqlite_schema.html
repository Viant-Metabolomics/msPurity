Table av_peaks

Table for averaged peaks. Used to record the averaged fragmentation spectra associated with for chromatography grouped peaks (e.g. XCMS grouped peaks). Three different methods can create the averaged spectra. "av_intra": averaged within files, "inter": averaged within files and then across files and "av_all": averaged across all associated spectra found for the chromatography grouped peak.

IndexesField NameData TypeDescription
* avid integer average peak id
  cl real cluster group - id given when clustering spectra
  mz real averaged mass / charge value
  i real averaged intensity value
  snr real averaged signal to noise ratio value
  rsd real Relative standard deviation
  count real Total number of peaks averaged across
  total real Total number of potential peaks that could be averaged across if each scan contained a corresponding peak
  inPurity real averaged precursor ion purity value
  ra real Averaged relative abundance
  frac real fraction of peaks (e.g. count/total)
  snr_pass_flag text flag for signal to noise threshold
  minfrac_pass_flag text flag for minfrac threshold
  ra_pass_flag text flag for relative abundance threshold
  pass_flag text overvall pass flag
  method text method used for averaging (av_inter, av_intra and av_all)
fileid integer fileid from fileinfo
grpid integer grpid from c_peak_groups
Indexes
pk_av_peaks ON avid
Foreign Keys
fk_av_peaks_fileinfo ( fileid ) ref fileinfo (fileid)
fk_av_peaks_c_peak_groups ( grpid ) ref c_peak_groups (grpid)

Table s_peak_meta

IndexesField NameData TypeDescription
* pid integer The "p" stands for purity as the id is calculated by msPurity
  seqNum real Sequence number. Similar to scan number but calculated by mzR at time of file parsing, so if scans have been removed prior to processing, the scan number and the seqNum will differ
  precursorIntensity real Precursor intensity as determined from original data file (e.g. mzML)
  precursorMZ real Precursor m/z as determined from original data file (e.g. mzML)
  precursorRT real Precursor rentention time as determined from original data file (e.g. mzML)
  precursorScanNum real Precursor scan number as determined from original data file (e.g. mzML)
  id real id (file specific)
  precursorNearest real Precursor scan number nearest to the fragmentation event as determined by msPurity
  aMz real The m/z value in the precursorNearest scan which most closely matches the precursorMZ value provided from the mzML file (msPurity)
  aPurity real The purity score for aMz
  apkNm real The number of peaks in the isolation window for aMz
  iMz real The m/z value in the precursorNearest scan that is the most intense within the isolation window
  iPurity real The purity score for iMz
  ipkNm real The number of peaks in the isolation window for iMz
  inPkNm real The interpolated number of peaks in the isolation window
  inPurity real The interpolated purity score
fileid integer fileinfo id
  acquisitionNum real
  retentionTime real
